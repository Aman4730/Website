Website React Project Structure aur Details:

1. Root Directory Files:
   - package.json: Project ki main configuration file
   - package-lock.json: Dependencies ki exact versions
   - .env: Environment variables
   - vite.config.js: Vite configuration
   - README.md: Project documentation
   - eslint.config.js: Code linting configuration
   - .gitignore: Git ignore patterns

2. Main Directories:
   
   a) src/ (Main source code):
      - App.jsx: Main application component (266 lines)
      - App.css: Main application styles
      - main.jsx: Entry point
      - index.css: Global styles
      
      Sub-directories:
      - Brochure/: Brochure related components
      - Imges/: Project images
      - components/: Reusable components
      - Pages/: Different pages/routes
      - api/: API related code
      - assets/: Static assets
      - PreLoader/: Loading screen components

   b) public/: Public static files
   
   c) dist/: Build output directory
   
   d) node_modules/: Project dependencies

Tech Stack:
- React (Frontend framework)
- Vite (Build tool)
- ESLint (Code linting)

Project Structure Overview:
- Modern React application with component-based architecture
- Uses Vite for fast development and building
- Organized code structure with separate directories for components, pages, and assets
- Environment variables support through .env file
- Git version control integration
